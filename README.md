# 我的简单网页项目

这是一个最简单的HTML页面项目，用于演示自动测试和部署流程，并包含访问统计功能。

## 项目说明

- `index.html` - 主页面文件
- 自动测试配置
- 自动部署配置
- 访问统计功能（本地计数器和Google Analytics）

## 访问统计功能

页面包含两层访问统计：
1. 本地访问计数器：使用localStorage记录访问次数
2. Google Analytics：用于详细访问分析（需要配置GA ID）
